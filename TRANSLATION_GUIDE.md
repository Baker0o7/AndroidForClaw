# Translation Guide

## Categories
1. **UI Elements**
   - Button Text
   - Navigation Labels
   - Menu Items

2. **Notifications**
   - Error Messages
   - Alerts
   - Warnings

3. **User Preferences**
   - Settings Descriptions

4. **Other**
   - Miscellaneous Strings

## Priorities
- High: Critical functionality and user interactions.
- Medium: Important but not critical.
- Low: Minor strings and descriptions.
